# tpf2-timetable-train-graph

This is tool for better understanding of timetable from game Transport Fever 2 mod Timetables 1.3

The tool shows timetable from line point-of-view such as station pov

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).
